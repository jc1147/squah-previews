Every site in this exhibition was built on SGEN itself. Same builder, same box, same baseline — nothing stitched together underneath.

01

One builder, no code

Each site is built in SG Builder — a visual, drag-and-drop builder with 48 components — from a finished template, not a blank page. Designers ship; developers optional.

02

Plugins already in the box

Forms, SEO, store, events, attribution, popups, backups — native modules, not plugins. No plugin tax, no suite tax, nothing to license or update against each other.

03

A free baseline under it all

The Foundation Pack ships free on every plan: hosting, auto-renewing SSL, global CDN, a WAF, automatic security patching, and the multi-site dashboard.
