Section block · Headers / Navs

Four header patterns, each a real section you can drop in

Every block below is a complete, production-grade <header> — responsive, copy-ready, and self-contained (system fonts, zero external dependencies, CSS-crafted visuals). Each ships the mobile hamburger → off-canvas drawer and the sticky behavior from the scroll library (the is-stuck boolean, F1 sentinel). Each demo scrolls inside its own frame so the sticky / transparent→solid behavior proves itself. Try the hamburger — narrow the window (or it auto-collapses inside the frame) and the drawer opens with scrim, scroll-lock and Esc-to-close.

01

Centered logo — nav split either side

When to use: editorial, fashion, hospitality, and personal brands where the wordmark is the brand. Symmetric, calm, premium. Best with a short, even nav (≤3 links per side). Shrinks the stacked logo to inline on scroll.

03

Transparent over the hero → solid glass on scroll

When to use: any page that opens on a dark or imagery hero — real estate, hospitality, events, automotive, premium product. The header floats transparent over the hero, then flips to solid glass the instant you scroll past it (one boolean). Never carry the transparent state into phone widths — it goes solid+static there.

04

Top utility-bar + main nav (two rows)

When to use: info-dense sites that need secondary affordances above the fold — clinics, law firms, local services, e-commerce, multi-region brands. A thin utility row (phone, hours, language, account) sits over the main nav, then collapses away on scroll so the main bar stays compact.
