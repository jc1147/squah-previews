Sections — Forms / Lead-gen

Forms & lead-gen sections

Five production-grade section blocks you can drop straight into a real page — two contact forms, two newsletter captures, and a multi-step lead-gen flow. Every control carries its full state machine (focus · error · success · disabled) drawn from the 07 · Components source of truth, all inputs are ≥16 px (no iOS auto-zoom), submits are ≥44 px tall, and each block reflows cleanly to a phone. One block carries a sticky summary rail that releases to static on mobile (per 08 · Scroll / Sticky).

Live & interactive — tab a field, trigger validation, run the multi-step flow. Self-contained: system fonts, zero external deps, imagery is CSS gradients.

BLOCK 03 · NEWSLETTER

Inline newsletter strip — dark band, pill submit

When to use: the conversion/closing band near the page foot, or a full-width interrupt between content sections. The email + submit sit on one row at desktop and stack at ≤767 (the pill drops below). High contrast on a dark ground makes it the loudest CTA on the page.
