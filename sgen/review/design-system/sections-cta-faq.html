Section blocks — the close: CTA + FAQ

CTA bands & FAQ sections

Six complete, copy-ready section blocks for the conversion tail of any page — four CTA bands and two FAQ layouts. Each is a real, full-width section you could lift straight into a build: live button states (fill / ghost / light-on-dark, with :hover / :active / :focus-visible / :disabled), a working inline form with validation states, and native <details> accordions with the accent active row. Built on the SGEN chrome — charcoal #1a1a1a, red #d51522, one 8 px button radius, ink-tinted shadows — zero external dependencies, system fonts, all imagery CSS-crafted, motion gated behind prefers-reduced-motion.

States work — hover any button, Tab for the focus ring, click an accordion row. Pattern IDs trace back to the composition + modern-design libraries.
