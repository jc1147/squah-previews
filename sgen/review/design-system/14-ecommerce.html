Page 14 — the e-commerce flow, live

E-commerce flow components

The full commerce set — shown live in every relevant state, not described. A PDP buybox (gallery with thumbnail rail + hover-zoom, color & size variant swatches, a real quantity stepper, and the add-to-cart state machine with wishlist), a cart with editable line-items + a sticky order-summary, a 3-step checkout stepper you can walk through, and an order-confirmation receipt. Built on the SGEN chrome: charcoal #1a1a1a, red #d51522, one 8 px button radius, the ink-tinted shadow family, motion gated behind prefers-reduced-motion. Self-contained — system fonts, CSS-crafted visuals, zero external dependencies.

Interactive states work — pick a swatch, step the qty, add to cart, walk the checkout, apply SGEN10. States marked PINNED are force-applied via a modifier class so one screenshot captures them.

The trust micro-row closes it out.
