From blocks to a template

3How to compose a template from blocks.

The blocks are the parts; the order is the architecture. The full procedure, the 16 DOM-verified per-vertical sequences, and the slot-atom legend all live in the composition library — this is the short version.

📐 source · _PAGE-COMPOSITION-LIBRARY.md → §0.2 (7-step procedure)

Pick the vertical

Look up its row in the library §2 — that gives the verified homepage sequence and inner-page blueprints.

Lay the universal spine

Drop the 8 zones (§1 above) as the frame. Every vertical IS this spine reordered and expanded.

Choose the hero TYPE

The most identity-defining slot — pick a block from the Heroes gallery (B1–B6) by vertical.

Drop in the vertical block

Zone 6 is the payload — Pricing, Gallery, menu or product grid — that makes it this vertical.

Apply the composition rules

How many sections, where proof sits, where the CTA lands, the vertical rhythm, and the never-do list (library §3).

Fill each slot + lay the through-lines

One accent colors the conversion path · alternating grounds are the skeleton · one transition family per page.

Reflow + clear the gate

Reflow every section per the responsive library and render-test at 390 + 768 + 1440. Fail-closed: an unproven breakpoint is a failing breakpoint.

The one-line heuristic

The spine is invariant; the hero TYPE and the vertical block are the two knobs that make a template read as a given vertical; the proof→pricing/FAQ→CTA tail is the close for conversion verticals and is swapped for a gallery/showcase tail on showcase verticals.

Three through-lines run the whole stack

What holds a composed page together.

These aren't blocks you place — they run across every block, top to bottom. Lay them once and a stack of independent sections reads as one designed page.

one accent

Color-code the path

A single hue paints every CTA, icon chip, counter numeral, link hover, active state and featured tier — the conversion path is the accent.

alternating grounds

Ground = skeleton

Full-bleed bands alternate paper → 3–8% tint → one near-black anchor. Depth is whitespace and figure/ground, not chrome between bands.

one transition family

Seam with intent

Shape dividers only where the mood fits; elsewhere color-band contrast + the image bottom-fade + inset panels. One family per page, 2–4 uses, never mixed.

reduced-motion

Gate every move

Every reveal, count-up and hover is gated behind prefers-reduced-motion — the page is fully legible and complete with motion off.
