Section blocks — Team & Gallery / Portfolio

Team & gallery sections

Five complete, production-grade section blocks — drop any one straight into a real page. Two team patterns (card grid + circular roster) and three gallery / portfolio patterns (masonry, filterable grid, hover-zoom tiles). Every block carries the SGEN imagery engine: image-zoom inside a clipped frame (H1 / TOP-12 #7), shape masks (H2), and the exposure hover (mask + rising caption + growing underline). Charcoal #1a1a1a, red #d51522, one radius language, ink-tinted shadows, motion gated behind prefers-reduced-motion. Imagery is CSS-crafted gradients — zero external images, zero dependencies.

Hover or tab-focus any portrait / tile — zoom, masks and caption-rise are live. Blocks marked S-TEAM / S-GALLERY map to the composition-library slot atoms.

A Team — card grid, hover-reveal S-TEAM G4 grid · H1 image-zoom · TOP-12 #7 reveal-bar

When to use: the standard "meet the team / our people" block on an About or Agency page. Portrait-first cards in a 4-up grid; on hover the card lifts, the portrait zooms, a social bar rises and a one-line bio expands — so the resting grid stays calm and scannable while detail is one hover away.

B Team — circular roster, hover-reveal bio S-TEAM H5 circular medallion · H1 zoom · H2 mask

C Gallery — masonry, hover-zoom + caption-rise S-GALLERY H1 `exposure` · CSS columns · G3 reflow

When to use: a portfolio / work / look-book grid where pieces have mixed proportions and you want a magazine, gallery-wall feel. CSS-columns masonry packs tall, wide and square tiles with no JS layout engine; hover plays the exposure motion — image zooms, a dark wash fades in, the caption rises and an accent underline grows. The default "selected work" pattern for agency and photography pages.

D Gallery — filterable grid, image-zoom + mask S-GALLERY G3 bento · H1 zoom + H3 overlay · filter (opacity)

When to use: a larger portfolio / case-study index where visitors self-select by category. Filter chips toggle tiles by fading (not removing) so the grid animates smoothly and never jumps; a category badge and title rise on hover over the zooming image. One bento "span-2" tile breaks the rhythm. Best for work indexes, product galleries and resource libraries.

E Gallery — hover-zoom tiles, circle-mask reveal S-GALLERY H2 circle mask · H1 zoom · G3 bento · dark stage

When to use: a tight, dense, dark-mode tile wall — photography sets, press shots, an Instagram-style grid, or a "more work" footer to a case study.
