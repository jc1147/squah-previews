Native module · Theme Editor

Theme Editor: your whole site's design, one panel.

SGEN's Theme Editor is a three-region design editor that sets your whole site from one place: a left rail with eight controls, a center panel that edits a 7-level type scale (Body plus H1 through H6, each a Font Family dropdown and a Font Size in px), and one green Save Changes that applies it everywhere.

Pick a named color swatch and it repaints colors, typography and button styles across the entire site in one click. The same Global Colors, Typography and Buttons tokens are editable inside the SG Builder Site Settings drawer without leaving the visual canvas. Type, color and button settings live together in one native panel, so you set the whole site's design in one place and Save once.

What you control

Eight controls. The whole site's design, in one form.

The Styles & Layout rail holds eight controls, and one Save applies them everywhere. These are global tokens, edited once in a native panel and never per template, so there is nothing to keep current on the side. Set typography, color, layout and buttons once, and the whole site follows.

Typography, site-wide

Set the body font and the size of every heading level. Seven rows: Body plus H1 through H6, each a Font Family dropdown and a Font Size in px. The live default scale runs Body 18, H1 60, H2 46, H3 36, H4 24, H5 20, H6 18.

Global Colors

A full token palette, not a few swatches: Body Background and Text plus accent, primary, secondary, success, info, warning, danger, light and dark, each as a Default and Hover pair.

Button Styles

Font, font size, border width, a four-corner border radius (set to 9999 for a pill), four-side padding, and Default and Hover overrides for text, background and border. Buttons are a first-class control here.

Themes and color swatches

Pick a full color swatch (Midnight Orchid, Slate Ember, Forest Mint, Burgundy Gold, Graphite Lime and more) to apply colors, typography and button styles site-wide in one click.

Native header and footer

Edit the global header (template, fonts, colors, logo alignment, sticky-on-desktop and sticky-on-mobile toggles, and in-header CTA buttons) and the footer (template plus a Top-Footer subsection with its own colors and HTML content) from the same rail. Plus a Mobile Menu editor.

Layout and accessibility

Theme mode (light or dark), container width (e.g. 1200px) and four-side section padding live in the Layout panel. An Accessibility ADA control sits in the same rail, alongside the controls you set type and color with.

Whole-site repaint

Pick a swatch, the whole site changes: color, type, buttons.

The Themes page is a gallery of named color swatches, each a mini site-preview thumbnail. Pick one and SGEN applies its colors, typography and button styles across the entire site at once. Switching a swatch repaints the hero, the CTAs, the cards and the footer in one Save, no per-page edits.

Self-hosted fonts

Upload your brand fonts. They flow into the type scale.

Custom Fonts is a self-hosted webfont manager. The Font Manage form holds a Title and one or more Weight blocks. Each Weight block carries a Font Weight select (normal plus 100 through 900, ten options) and four upload slots, TTF, WOFF, WOFF2 and EOT, each with real browser-coverage helper text. Add Another stacks more weights, so one family ships its 300, 400 and 700 under a single font-family. Publish it (or stage it as Draft), and a published family auto-injects an @font-face block into every public page's head with font-display: swap, then becomes selectable in the Typography Font Family dropdowns.

Custom code, native

A managed snippet library, not a paste box.

Custom Codes manages your HTML snippets, scripts, pixels and tracking tags. Each snippet carries a placement (head, body start, or body end), a priority for deterministic ordering, and an Active or Inactive status, with a Trash and Restore safety net. Its sibling Custom CSS holds site-wide stylesheet snippets the same way, both in the native Appearance group.

Questions, answered

The Theme Editor, in plain terms

Do global style changes apply everywhere at once? +
Yes. One Save in Styles & Layout applies typography, Global Colors, Layout and Button Styles site-wide. Change the 7-level type scale (Body plus H1 to H6) or the 10-token Default and Hover palette once, and the whole site updates from those global tokens, with no per-template edits.
Can I use my own brand fonts? +
Yes, self-host them. Upload TTF, WOFF, WOFF2 and EOT files per weight (normal plus 100 through 900), stack multiple weights under one family with Add Another, set the font to Publish, and the family becomes selectable in the Typography Font Family dropdowns and injects with font-display: swap. Custom Fonts hosts your own licensed files. It does not manage Google Fonts here, and OTF and variable fonts are not supported.
Can I switch the whole site's look quickly? +
Yes. Pick a named Color Swatch, such as Midnight Orchid, Slate Ember, Forest Mint, Burgundy Gold or Graphite Lime, and SGEN applies its colors, typography and button styles across the site in one click. There are five named swatches and more below the fold.
Can I edit the design while building a page? +
Yes. The SG Builder Site Settings drawer edits the same Global Colors, Typography and Buttons tokens inside the visual canvas, grouped as Design System, Theme Style and Settings, with responsive breakpoint chips from 1920 down to 480px. The admin Theme Editor uses a green Save, the drawer uses a white one, and both write the same global tokens.
What does it not do? +
There is no preview-before-publish stage for global Appearance edits, changes go live on Save. The Typography panel exposes Font Family and Font Size only, with no weight or line-height columns. Custom Fonts hosts your own files, so it does not manage Google Fonts here, and OTF and variable fonts are not supported. We would rather be plain about the edges than over-claim them.
Is it a plugin I install, and which plan includes it? +
No separate plugin. The Theme Editor is one of 23 native modules in SGEN, on every plan, which differ only by how many live sites you run. Paid plans start at $39/mo (billed annually, $47 monthly), and the free Sandbox runs it too, so you can design your whole site before you pay anything.
