The asset hub

Every asset in one place, with optimization baked in.

The library is the central home for every image, video, document, and downloadable file on your site. Browse in grid or list view, filter by type, sort by date, name, or file size, and search by filename or uploader. Each file carries four inline actions: Preview, Edit, Copy Link, and Delete.

Upload and reuse

Files uploaded here are available everywhere: pages, posts, products, and the blog all pick from this same library. Upload once, pull the same asset anywhere, and stop re-uploading duplicates.

WebP / AVIF on upload

Turn on Compress + Convert to WebP and images are converted and compressed as they land, so they load fast for visitors. A multi-megabyte photo drops to a few hundred kilobytes with no visible quality loss.

AI media captions

Optionally auto-generate a caption on new image uploads for accessibility and search, then review and refine in the library. Treat the suggestion as a first draft you approve, not auto-text you have to trust blindly.

Edit alt text once

Change alt text, captions, and display names in one place and the correction ripples to wherever the file is displayed across the whole site.

Copy public link

Grab any file's public URL in one click, ready to paste into an email, a campaign, or anywhere off-site.

Bulk delete, safely

Select multiple files, delete, and confirm in a modal that lists every file first, so you can clear out clutter without nuking an asset that is still in use.

Questions, answered

Media Library, in plain terms

Does Media Library handle WebP automatically? +
Yes. The library converts and compresses images to WebP/AVIF. Turn on Compress + Convert to WebP at upload and a multi-megabyte photo lands at a few hundred kilobytes with no visible quality loss. It is native, with no optimization plugin to install.
Is Media Library available on the free Sandbox? +
Yes. Media Library is part of SG-Core, included in every plan, including the $0 Sandbox. The library itself, with WebP/AVIF optimization, is included everywhere. Serving media from a Google Cloud bucket is an optional cloud-storage add-on on most plans.
Can it generate alt text or captions for me? +
Optionally, yes. Turn on AI Media Captions in Media Settings and new image uploads get a suggested caption you can review and refine. Treat the suggestions as a first draft: they speed up accessibility and search hygiene, and you keep editorial control. It is a head start you approve, not auto-text you have to trust blindly.
Where are my files served from? +
From the local server by default. On accounts with the cloud-storage add-on, you can switch to a Google Cloud bucket. Run the one-time sync first so images do not break during the switch.
Do I still need a separate media or optimization plugin? +
No. Upload, organization, search, WebP/AVIF optimization, alt text, and optional AI captions are all native to the Media Library. Fix an alt text in one place and it is correct across the whole site, grab a direct asset link instantly, and clean out old files in bulk behind a confirmation that lists every file first. There is no plugin to license or keep current for any of it.
