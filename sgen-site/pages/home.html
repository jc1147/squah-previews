Before you start

Questions, answered

How do I get started, and do I need a credit card? +
You can start on the free Sandbox with no credit card and no time limit. Build a real site, explore the modules, and only move to a paid plan when you are ready to go live.
What does it cost? +
Paid plans start at $39/mo (billed annually; $47 monthly), and every plan is the whole platform: all 22 native modules and the free Foundation Pack. SGEN prices by how many live sites you run, not by feature: Indie (1) $39/$47, Studio (3) $97/$117, Scale (10) $297/$333, Enterprise (unlimited) $497/$577. There is a 2-week trial, and you can cancel anytime, prorated. See the full breakdown on the pricing page.
Can I move my existing WordPress site over? +
Not as a one-click import, and we will not pretend otherwise. Moving to SGEN is a rebuild: either we rebuild your site for you (white-glove) or you rebuild it yourself with the builder and templates. You leave the plugin debt behind in the process, which is usually the point. See the migration page for both paths.
Is it really secure without all my plugins? +
That is the security model, not a gap. 91% of WordPress vulnerabilities were in plugins (Patchstack, State of WordPress Security 2026). SGEN ships features natively, so there is no stack of third-party plugin code to be your weakest link, and a WAF plus security patching are included free on every plan.
Do I have to learn a whole new way to build? +
If you have used Elementor, the builder will feel familiar: the same drag-and-drop precision, without the plugin underneath. Templates start new pages finished instead of blank.
What about HIPAA or other compliance requirements? +
The Foundation Pack includes a WAF, SSL, and security patching on every plan, and SGEN keeps a per-session consent log for audit. Dedicated HIPAA compliance is coming soon. For specific regulatory requirements, talk to sales.
