Questions

Questions about the guides

How long are the guides? +
Most run 5 to 15 minutes. Every guide is step-by-step: it starts where you are and ends at a working result, with no prerequisite beyond a free Sandbox account.
Are the guides kept up to date? +
Guides are reviewed for accuracy against the current platform. If something looks out of date, the feedback link on each guide routes it to us.
Which plan do I need to follow the guides? +
Most guides work on the free Sandbox ($0). Every guide covers features in the all-inclusive platform, which is the same on every plan, so going live just means picking a plan by site count (paid plans from $39/mo, billed annually; $47 monthly). The Sandbox is always the right place to start.
Are there guides for agencies running multiple client sites? +
Yes. The "Run multiple locations" and "Getting started with SG Builder" categories cover multi-site workflow: the Site Manager dashboard, login-as-client, and role-based access control. Browse those categories above.
Do the guides assume I already have a site to move over? +
No. Each guide stands on its own and never assumes you are bringing an existing site. If you are rebuilding an older site, note that SGEN is a rebuild, not an import: the rebuilding category covers both the white-glove and DIY paths.
